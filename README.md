# Install
## Vim package
```
git clone https://github.com/Gakgu/FocusCenter.vim.git ~/.vim/pack/plugins/start/FocusCenter
```

## Vundle
Add your `.vimrc`
```
Plugin 'gakgu/FocusCenter.vim'
```

# Usage
- `:FocusCenterOn`
  - Activate FocusCenter
- `:FocusCenterOff`
  - Deactivate FocusCenter
- `:FocusCenterToggle`
  - Toggle FocusCenter
